Google Scholar Report Generator

How to Use:

1. Click here to download the .zip file containing the necessary files.
2. Extract to your machine using your favorite decompression utility. (Right Click the .zip ---> Extract Here)
3. Open urls.txt, modify the pre-populated text document in the format provided, and save.
4. Open GSRG.jar, choose your formatting options, and click "Generate". A file will be generated in the local directory and automatically opened.

Notes:

  • If the program fails to display a single professor's publications, check that the URL associated with that professor is working in urls.txt. Then try it again.
    		•
  • If the program fails to display any publications in the report, Google may have changed their Scholars layout which may have broken the program. E-mail me and I'll update it to work with the current version of Google Scholar.

    Watson Videos Page

    How to Update:

    1. Go to the Watson Research Videos webpage. Scroll to the bottom and click "Last Updated: XX/XX/XX" and sign in.
    2. Upload a 250x200 thumbnail of the video you would like to add to /watson/research/videos/images. Add an additional layer to your thumbnail using YouTubePlay100x100.png (located in /images).
    3. Return to index.pcf and edit one of the three columns (depending on which is next to be filled). View the HTML for that column, and copy an existing block with the Javascript function I created. Edit all the arguments (name, description, YouTube URL, alt text, etc.) and save.

    Notes:

  • Make sure that when you click the thumbnail, it brings you to the top of the page and updates all of the information in the area to the right of the video.

    Watson Publications Page

    How to Update:

    1. Go to the Watson Publications webpage. Scroll to the bottom and click "Last Updated: XX/XX/XX" and sign in.
    2. Open the Google Scholar Reports Generator (see tutorial above) and generate a report using the "Binghamton Publications Webpage" output format.
    3. Copy the HTML of the webpage generated. Edit the main content area and paste your clipboard into the HTML Editor. Modify the document as need be from there.

    Watson Stats Dashboard - Version 1.1

    How to Use:

    1. Click here to download the .jar file.
    2. Open the file, click "Browse" and select your input data spreadsheet.
    3. Click "Create Dashboard" and a the dashboard spreadsheet should automatically open.

    Notes:

  • If there are any major changes to the layout of the input spreadsheet, the program may not work. Let me know of any changes to the input spreadsheet if the program fails.
